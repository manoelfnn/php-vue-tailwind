#PHP, Vue.Js e Tailwind 
Simples projeto que consume uma API. O projeto utilzia a seguinte stack de tecnologias:
PHP, Vue.JS 3 e Tailwind.

##Instalação
- Instale o docker;
- Faça um Pull deste repositório;
- Crie a imagem do projeto. 
Dentro do diretório do projeto, execute: ``docker build . -t php-vue-tailwind``
- Execute a imagem: ``docker run -d -p 8080:8080 php-vue-tailwind``
- Abra o navegador e acesse: ``http://localhost:8080``
